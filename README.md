# webmaster
HI!!!
This site was made by I and a classmate for the TSA competition Webmaster, for a community located near us! It includes tons of resources for our area, with a search feature to sift through them, information about the community itself, and some other general info about the area. It's pretty awesome, and has a great UI too! We reused about half of the code from our old website, Jobees, to save time and fit the TSA scoring standards better.

## Documentation
We documented our work on this site through worklogs, in the reference page! We also used commits.
